# egl-frontend

This is the frontend UI for the EGL Budget Monitoring System application built using **React**.

### Prerequisites
- Node.js (v14 or higher)
- npm

### Steps to setup

#### 1. Clone the Repo and install dependencies

```bash
git clone https://github.com/Velu-profile/EGL-frontend.git
cd egl-frontend
npm install
```


#### 2. Setup the environment variables

```bash
Example:
REACT_APP_BACKEND_DOMAIN=
REACT_APP_SUPABASE_URL=
REACT_APP_SUPABASE_ANON_KEY=
```


#### 4. Run the Application

```bash
npm run start
```

The application will run at: [http://127.0.0.1:3000](http://127.0.0.1:3000)
